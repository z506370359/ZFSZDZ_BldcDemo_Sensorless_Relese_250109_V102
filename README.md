# 🚀 项目详细说明书：无感方波无刷直流电机（BLDC）控制系统

## 📋 一、 项目简介
* **项目名称**：张飞实战电子“扬帆起航”课程无感方波控制。
* **核心功能**：使用静态脉冲注入法实现电机启动并切换至无感运行状态。系统支持灵活的启动方式，可以选择直接切无感，或者在升频升压阶段后切无感。
* **主控芯片**：系统基于 STM32G030 系列单片机开发。

## ⚙️ 二、 硬件与系统配置参数
* **系统主频**：单片机主频设置为 64 MHz。
* **PWM 配置**：
    * PWM 模块工作频率为 16 KHz。
    * 正常占空比调节范围在 15% (300) 到 25% (500) 之间，步进值为 1。运行极限占空比限制范围为 10% (200) 到 1000% (2000)。
* **母线电压 (Vbus) 采样网络**：硬件分压电阻比例为 20K / 2K，依托 3.3V 内部基准和 12 位 ADC (4096) 精度进行数值转换。
* **温度采样**：集成 NTC 温度传感器采集真实温度值。

## 🎛️ 三、 电机运行参数
* **转速范围**：系统设定电机的最小运行速度为 350 RPM，最大运行速度为 3000 RPM。
* **旋转方向**：支持顺时针 (CW) 和逆时针 (CCW) 旋转，系统默认方向为逆时针转动。
* **调速模式**：默认使用电位器旋钮进行调速设定，同时也支持切换为串口 (USART) 通讯调速。
* **运行模式**：支持开环运行与闭环运行，系统启动后默认工作在闭环工作模式。

## 🔌 四、 核心控制策略与算法
* **初始定位与启动**：
    * 预定位占空比设定为 5%~10% (对应内部参数值 200)，预定位时间为 10ms。
    * 系统使用 IPD (Initial Position Detection) 进行转子扇区判定，通过记录各绕组打脉冲对应的电流值并进行叠加（如 SumIab_Iba 等计算）来确认初始位置。
* **强拖换相配置**：换相时间对应 300 RPM 至 600 RPM 的速度区间，调节步进值为 10。
* **无感反电动势 (BEMF) 检测**：基于 1/2 Vbus 进行反电动势过零比较，理论判定系数为 50% (可微调)。配套有过零点计数器、端电压阈值设定与退磁时间补偿处理。
* **PI 闭环控制算法**：具有比例系数 (Kp)、积分系数 (Ki) 以及抗积分饱和系数 (Kc) 的速度闭环 PI 调节器模块，提供上下限幅以确保输出稳定。

## 🛡️ 五、 状态监控与保护机制
配置有全面的电机运行故障保护体系：
* **过流保护**：具备硬件过流计数和软件过流计数双重保护。
* **电压保护**：实时监测输入电压，具备过压计数和欠压计数保护。
* **运行状态保护**：包括电机过温保护、堵转电流保护、堵转相位保护以及启动失败保护处理。

## 🖥️ 六、 交互与可视化通信
* **LCD 数据显示屏**：支持彩屏显示，定义了白、黑、红、蓝、浅绿、深蓝等数十种颜色宏以绘制复杂的窗体背景及数据曲线。
* **USART 串行通讯**：系统能通过串口进行开环占空比设定、闭环目标速度设定，包含完整的接收与发送请求标志和校验位处理。
* **VOFA+ 调试上位机支持**：内嵌针对 VOFA+ 上位机的通讯协议结构体，允许通过浮点数据帧进行波形监控与参数调试分析。
